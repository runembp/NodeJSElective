Welcome to DankMemesAPI.

For usage, please refer to the following endpoints:

/dankmemes - GET - Retrieve all dank memes

/dankmemes/{id} - GET - Retrieve dank meme by id

/random - GET - Retrieve a random dank meme

/dankmemes - POST - Create a new meme

/dankmemes/{id} - PUT - Overwrites a meme based on the id

/dankmemes/{id} - PATCH - Changes an attribute of a meme based on the id

/dankmemes/{id} - DELETE - Deletes a meme based on the id
